
# Power BI Template Placeholder

This folder contains starter assets for Power BI. Due to proprietary file formats, a `.pbix/.pbit` file is **not auto-generated** here. 

## Quick Start
1. Open Power BI Desktop.
2. Get Data → Text/CSV → load files from `data/raw/`.
3. Build model: relate `bom_products` ↔ `licenses` ↔ `vendors`.
4. Recreate visuals per `docs/kpi_definitions.md`.
5. Save as `BOM_Software_Compliance.pbix` in this folder.

*Included:* `theme.json` for styling.
